# Tixano


Tixano is a Cardano-powered NFT ticketing platform focused on making event registration and verification more secure, transparent, and user-owned.

Instead of traditional tickets locked inside centralized systems, Tixano gives attendees NFT tickets that live in their wallets and can be verified on-chain.

---

# 🌍 Why Tixano?

Most event ticketing systems today are still built around centralized databases and temporary access codes.

That creates problems:
- Fake tickets
- Ticket duplication
- Scalping abuse
- Poor ownership experience
- Lack of transparency

Tixano reimagines event ticketing using blockchain infrastructure and wallet-based verification.

Tickets become digital assets users actually own.

---

# 🚀 What Tixano Does

With Tixano, event organizers can:
- Create and manage events
- Issue NFT tickets to attendees
- Verify attendees securely during check-in

Attendees can:
- Register using their Cardano wallet
- Receive NFT tickets directly in-wallet
- Check into events using QR verification
- Receive proof-of-attendance NFTs

---

# ✨ Core Features

## 🎟️ NFT Ticketing

Every event on Tixano generates its own NFT ticket collection.

Each ticket is:
- Unique
- Verifiable on-chain
- Linked to a real event
- Stored directly in the attendee’s wallet

---

## 🔐 Wallet-Based Authentication

No usernames.  
No passwords.  
No traditional accounts.

Users authenticate directly with their Cardano wallet.

---

## 📲 QR Code Check-In

Every NFT ticket contains a unique ticket identifier tied to its owner.

At event check-in:
- QR code is scanned
- Ticket validity is verified
- Ownership is confirmed
- Ticket usage status is checked

Once verified, access is granted instantly.

---

## 🏆 Attendance Proof NFTs

**Planned for future releases.** After successful check-in, attendees will receive attendance NFTs as proof they participated in the event.

This will create verifiable on-chain event history and participation records.

---

## ⚙️ Event Controls

Organizers can configure:
- Registration deadlines
- Event access flow
- Ticket distribution

Paid ticket support is planned for future releases.
